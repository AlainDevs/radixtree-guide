Debian installation
> https://openresty.org/en/linux-packages.html#debian
> 
> wget https://github.com/api7/lua-resty-radixtree/archive/refs/heads/master.zip
> 
> unzip master.zip
> 
> cd lua-resty-radixtree
> 
> make

Start:
> ./start.sh

Stop:
> ./stop.sh
